# 🎯 Real-Time Face Recognition System

A Python-based real-time face recognition system using OpenCV and machine learning.  
This project can detect and recognize human faces through a webcam.

---

## 🚀 Features
- Capture face images from webcam  
- Train model on captured data  
- Recognize faces in real-time  
- Fast and efficient processing  

---

## 🛠️ Tech Stack
- Python  
- OpenCV  
- NumPy  
- Streamlit (optional)

---

## 📂 Project Structure
face-recognition/
│── app.py  
│── capture.py  
│── model.py  
│── recognize.py  
│── dataset/  
│── model/  
│── requirements.txt  
│── haarcascade_frontalface_default.xml  

---

## ⚙️ Installation

git clone https://github.com/Arjunyadav47976/Real-Time-Face-Recognition-System.git  
cd face-recognition  

pip install -r requirements.txt  

---

## ▶️ Usage

python capture.py  
python model.py  
python recognize.py  

(Optional UI)  
streamlit run app.py  

---

## 📸 How It Works
- Detects faces using Haar Cascade  
- Stores face data  
- Trains a model  
- Recognizes faces in real-time
